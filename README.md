# Mendelian-randomization-Project
Here you will find code and tips for causal inference analyses using Mendelian Randomization, in the following manuscript: 


Grosso et al 2025: Causal relationships between gut microbiome and hundreds of age-related traits: evidence of a replicable effect on ApoM protein levels. Published on Aging [https://doi:10.18632/aging.206293](https://doi.org/10.18632/aging.206293) 

You can also see our preprint version at https://www.medrxiv.org/content/10.1101/2025.02.03.25321568v1 

or a YouTube video explaining the study in summary: https://www.youtube.com/watch?v=qYg42_gn_pw
